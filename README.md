# faq-leyu-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容来源于互联网上公开可获取的信息，以静态文件的形式在此处进行整理与备份。

## 项目简介

faq-leyu-archive 是一个纯静态归档仓库，主要存放一系列 HTML 文档。  
仓库本身不指向或服务于任何特定域名、网站或平台，仅作为信息保存与访问的辅助空间。

## 目录说明

- `./` 根目录下直接存放各独立 HTML 文件，文件命名通常反映其内容主题。
- 部分页面可能引用本地资源（如 CSS、图片），相关资源文件存放在同目录或子目录中。
- 无额外构建脚本或依赖，所有页面均为可直接在浏览器中打开的标准 HTML 文件。

## 页面归档说明

- 归档内容以 HTML 格式保存，尽量保留原始页面结构与可读性。
- 不保证所有页面均与原始来源完全一致，可能存在格式适配或精简。
- 归档不涉及任何形式的用户数据收集或追踪，仅提供静态内容展示。

## 维护说明

- 本仓库不定期更新，新增页面或补充已有内容。
- 如发现页面显示异常、链接失效或内容错误，欢迎提交 Issue 说明情况。
- 不保证所有历史页面长期可用，可能因内容调整而移除部分文件。

## 使用方式

- 直接访问仓库内的 HTML 文件即可查看对应页面内容。
- 可克隆或下载仓库到本地，通过浏览器打开 HTML 文件进行离线浏览。

## 许可

本仓库内 HTML 页面内容版权归其原始创作者所有。  
本仓库仅作存档与分发用途，不主张对归档内容的任何所有权。
